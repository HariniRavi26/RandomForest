# 💼 Salary Prediction Using Random Forest

This project predicts employee salaries using the Random Forest Regression algorithm based on historical data. The model was trained, evaluated, and deployed using Streamlit to provide an interactive salary prediction application.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- NumPy
- Joblib

## Features
✔ Data preprocessing  
✔ Random Forest model training  
✔ Salary prediction  
✔ Model evaluation  
✔ Streamlit deployment  

## Files
- app.py  
- salary_model.pkl  
- requirements.txt  
- dataset.csv
